# 🔨 TEKKIN: The Iron Mallet Tournament (鉄琴) 🔨

TEKKINはブラウザ上で鉄琴をセッションできるゲームです。  
（物理モデリング音源と WebRTC を無駄に極限チューンした、超銀河低遅延の鉄琴バトルアリーナでもあります）

👉 **[GitHub Pages でアリーナへ入場する](https://jamylas.github.io/TEKKIN/)**

---

## 🪵 特異な3種のマレット（バチ）
叩くマレットによって音色と見た目がガラリと変化します。己の戦闘スタイルに合ったものを選びましょう。

- **木製マレット (Medium Wood)**:  
  王道のバランス型。程よい硬さで、もっとも無難に美しい音を奏でます。初心者はまずこれ。
- **真鍮製マレット (Hard Brass)**:  
  極めて硬い金属のバチ。脳髄をダイレクトに揺さぶる鋭い金属音を放ちます。相手の鼓膜を攻撃したい時に。
- **ゴム製マレット (Soft Rubber)**:  
  柔らかいゴム製のバチ。優しく丸っこい「ポコポコ」サウンドを放ちます。深夜のステルス合奏に最適。

> 💡 **鉄琴の物理法則（打鍵位置による音色変化）**:  
> 音板の**中央**を叩くと完璧で豊かな残響が得られますが、**固定ネジ周辺**を狙い撃つと振動がミュートされ「コッ…」という虚無のアタック音が鳴り響きます。また、**音板の両端**を叩くと金属の悲鳴のような超高域倍音が炸裂します。

## 🏟️ 3つの合奏モード（オンラインアリーナ）
WebRTC P2Pスター接続により、世界中の鉄琴使いと低遅延でセッション（殴り合い）が可能です。

- **公開アリーナ (Grand Arena)**:  
  入室した瞬間、全員が自動的に一つの巨大な公開アリーナ（`TEKKIN-GRAND-ARENA-JAMYLAS-V2`）に接続されます。見知らぬ鉄琴戦士たちと即興のセッションを繰り広げましょう。
- **野良合奏 (Quick Match Bands)**:  
  最大4人の小規模アリーナへ自動でマッチングされます。もしホストが突然切断されても、残されたゲストが瞬時にホストへ昇格する「ゾンビ・フェイルオーバーシステム」を極秘裏に搭載。
- **非公開ルーム (Private Room)**:  
  ルームを作成してURLを共有すれば、特定の友人とだけ密室でこっそり鉄琴を突き合わせることができます。

## 🎙️ 録音機能 ＆ メトロノーム（支援兵器）
孤独なトレーニングや演奏分析に役立つ高機能ツール。

- **BATTLE RECORDER (演奏録音)**:  
  己の超絶技巧をミリ秒単位で完璧に記録し、その場でプレイバック（自己満足）できます。
- **BEAT METER (高精度メトロノーム)**:  
  Web Audio APIの正確な時間軸と連動し、LEDフラッシュ同期で脳内に一定のテンポを刻み込みます。

## 🎹 簡易操作マニュアル

- **マウス / トラックパッド**: 音板をクリック、またはドラッグ（スイープ奏法）で連打。
- **キーボード (PC専用)**:
  - **白鍵（全音）**: `A` `S` `D` `F` `G` `H` `J` `K` `L` `;` `'` `]` (C5〜G6に対応)
  - **黒鍵（半音）**: `W` `E` (空白) `T` `Y` `U` (空白) `O` `P` (空白) `[` (C#5〜F#6に対応)
  *※超高音域（G#6〜C8）はマウスで直接ヒットしてください！*
- **タッチ操作**: モバイル端末のマルチタッチで和音の連打に対応。
