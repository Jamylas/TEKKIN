## 🌌 『TEKKIN』とは？

かつて『星空の鉄琴』と呼ばれ、きらきら星を練習していた可憐なアプリは、度重なるアップデートと熱狂的プレイヤーによる接続負荷（ラグ）の荒波に揉まれ、無駄な装飾を削ぎ落とした**超実戦型音楽戦闘用デバイス**へと進化を遂げました。

## 🌟 狂気的なまでの機能と特徴

### 1. ガチすぎる物理モデリング音響 (Web Audio API)
外部の重たい音源ファイルのロード待ちなどという軟弱な仕様はありません。基本波＋非調和倍音（2.76倍、5.4倍、8.93倍）および打撃ノイズをリアルタイム演算で叩き出します。
- **打鍵位置によるシビアな音色変化**:
  - **中央部分 (0.5)**: 完璧なヒット。豊かな残響と芯のある美しい音が空間に響き渡ります。
  - **ネジピン周辺 (0.224 / 0.776)**: 物理的な「節（ノード）」。振動が強制的にミュートされ、「コッ…」という虚無のアタック音だけが残ります。嫌がらせ演奏に最適。
  - **音板の両端 (0.0 / 1.0)**: 金属の悲鳴。鋭くキンキンした高域倍音とマレットの衝撃音が響き渡り、相手の精神を揺さぶります。

### 2. 鍵盤数のインフレ (37鍵・C5〜C8)
以前の全音域から大幅に拡張され、なんと **37鍵（C5〜C8）** を網羅。人類の可聴領域の限界に迫る超高音（C8：約4186Hz）で、相手の三半規管にダイレクトアタックが可能です。

### 3. 三大マレット（バチ）の使い分け
選択するマレットによって、叩いた時の物理シミュレーションと見た目が連動します。
- 🪵 **木製 (Medium Wood)**: 王道のバランス型。
- 🪞 **真鍮製 (Hard Brass)**: 脳髄を貫くメタリックサウンド。
- 🪳 **ゴム製 (Soft Rubber)**: 優しく包み込むポコポコサウンド。

### 4. 三大オンライン合奏アリーナ (WebRTC / PeerJS)
P2Pスタートポロジー接続により、世界中の奏者とのセッションを実現。
- 🏟️ **公開アリーナ (Grand Arena)**:  
  ページを開いた瞬間、強制的に同一アリーナ（`TEKKIN-GRAND-ARENA-JAMYLAS-V2`）へ放り込まれます。見知らぬ誰かと静寂の鉄琴バトルを繰り広げましょう。
- 👥 **野良合奏 (Quick Match Bands)**:  
  最大4人のバンドルームへ自動マッチング。ホストが突然失踪（回線切断）しても、残されたゲストが瞬時にホストへと自動昇格する**ゾンビ・フェイルオーバーシステム**を搭載。
- 🔒 **非公開ルーム (Private Room)**:  
  秘密の部屋を作成し、URLを送りつけて特定の相手とだけ密室で叩き合えます。

### 5. 充実の支援兵器
- **BATTLE RECORDER (演奏録音)**: 己の超絶演奏をミリ秒単位で正確に記録・再生可能。
- **BEAT METER (高精度メトロノーム)**: Web Audioの正確なクロックと連動するLEDフラッシュ付きメトロノーム。
- **宇宙のエフェクト**: 「宇宙の残響（ステレオリバーブ）」と「やまびこ（ディレイ）」で、音響空間を汚染できます。

## 🎹 操作マニュアル

- **マウス/トラックパッド**: 音板をクリック、またはドラッグによるスライド（スイープ）奏法で連打。
- **キーボード (PC専用ショートカット)**:
  - **白鍵（全音）**: `A S D F G H J K L ; ' ]` (C5〜G6に対応。超高音域はクリックで！)
  - **黒鍵（半音）**: `W E` (空白) `T Y U` (空白) `O P` (空白) `[` (C#5〜F#6に対応)
- **タッチ操作**: モバイル端末のマルチタッチで和音を狂ったように連打可能。画面破壊に注意。
