# Celestial Glockenspiel (星空の鉄琴) ✦

ブラウザ上で美しく直感的に演奏できる、高機能な鉄琴（グロッケンシュピール）アプリケーションです。宇宙や星空をモチーフにしたダークモードの洗練されたビジュアル、Web Audio APIによる本格的な物理モデリング音源、キーボード操作、録音・再生、そして楽譜に合わせた演奏練習モードを提供します。

---

## 🌟 主な機能と特徴

1. **美しい星空テーマのUI/UX**:
   - 深みのある宇宙グラデーション背景と明滅する星々。
   - ガラスモーフィズム（すりガラス効果）を取り入れた操作パネル。
   - 金属の立体的な光沢、反射、固定ピン（振動の節）を再現した音板デザイン。
   - 打鍵時に発生するCanvasによる星屑のパーティクルエフェクトと同期ビジュアライザー。
   - 選択された素材（木製・真鍮製・ゴム製）に連動する3D風マウス追従マレット（クリックで叩き込みアクション）。

2. **本格的な物理モデリング音響 (Web Audio API)**:
   - 外部音声ファイルのロード遅延なし。基本波＋非調和倍音（2.76倍、5.4倍、8.93倍）およびアタックノイズを合成してリアルな鉄琴の響きを再現。
   - **打鍵位置による音色変化の再現**:
     - *中央部分 (0.5)*: 基本音（芯のある豊かな音）が最大化され、豊かな残響が得られます。
     - *固定ネジ周辺 (0.224 / 0.776)*: 振動の「節」にあたるため音が大きくミュートされ、アタック音中心の短い音になります。
     - *音板の両端 (0.0 / 1.0)*: 金属的な高い非調和倍音とマレットの衝撃音が強調され、鋭くきらびやかに響きます。

3. **豊富なエフェクト & 高精度メトロノーム**:
   - ホワイトノイズの指数減衰を利用したステレオリバーブ（宇宙の残響）と、フィードバックディレイ（やまびこ）を搭載。
   - Web Audio APIの正確な時間軸で動作する、音ズレの一切ないメトロノーム（LEDフラッシュ同期）。

4. **演奏練習 (プレイアロング) モード**:
   - 「きらきら星」「さくらさくら」「歓喜の歌」「ふるさと」を収録。
   - 音符（星）が上から降ってくるガイドに合わせ、次に叩くべき鍵盤が点滅し、ユーザーが押すまで待ってくれるガイド待機機能付き。

5. **録音・再生**:
   - 演奏をミリ秒単位で正確に記録し、その場でプレイバック可能な録音機能を搭載。

## 🎹 操作方法

- **マウス**: 音板をクリック、またはドラッグによるスライド演奏。
- **キーボード (PC)**:
  - **白鍵（全音）**: `A S D F G H J K L ; ' ]` (C5〜G6に対応)
  - **黒鍵（半音）**: `W E` (空白) `T Y U` (空白) `O P` (空白) `[` (C#5〜F#6に対応)
- **タッチ操作**: スマートフォンやタブレットでのマルチタッチ（同時打鍵・和音）に対応。
